# Week-3-module
This repository contains the Java solutions for the Week 3 Algorithms Lab. The programs are implemented according to the given assignment requirements and demonstrate the working of three important sorting algorithms:

Merge Sort
Quick Sort
Heap Sort

Each program is written in Java and is compatible with Visual Studio Code and other standard Java IDEs.

Requirements
Java JDK 8 or above

How to Run
Clone the repository:
git clone (https://github.com/AKSHITA-AK/Week-3-module)
Open the project in Visual Studio Code.
Compile the Java file:
javac FileName.java

Example:

javac MergeSortEasy.java
Run the program:
java FileName

Example:

java MergeSortEasy
Enter the input when prompted.

Learning Objectives
Understand Divide and Conquer techniques
Implement Merge Sort recursively
Implement Quick Sort using partitioning
Implement Heap Sort using Max Heap
